## Why GitDesk?

Stop context-switching to GitHub. GitDesk brings your PR review queue to your desktop with instant notifications, so you never miss a review request again.

## Features

<CardGrid stagger>
  <Card title="Unified PR Inbox" icon="approve-check-circle">
    All your review requests, created PRs, and recently reviewed PRs in one clean interface. No more hunting through GitHub notifications.
  </Card>
  <Card title="One-Click GitHub OAuth" icon="github">
    Sign in securely with GitHub's device flow. No tokens to create, copy, or manage. Just click and authorize.
  </Card>
  <Card title="Organization Filter" icon="setting">
    Working across multiple orgs? Filter your inbox to focus on a single organization at a time. Your selection persists across sessions.
  </Card>
  <Card title="Desktop Notifications" icon="bell">
    Native Linux notifications alert you the moment you're requested to review a PR. Never leave your teammates waiting.
  </Card>
  <Card title="Lightning Fast" icon="rocket">
    Built with Flutter for native Linux performance. Cached data means instant loads, with smart background refresh every 5 minutes.
  </Card>
  <Card title="Open Source" icon="open-book">
    MIT licensed and built in the open. Check out the code, report issues, or contribute on GitHub.
  </Card>
</CardGrid>

## Quick Start

```bash
# Download the Flatpak from GitHub Releases
flatpak install GitDesk-*.flatpak

# Launch and sign in with GitHub
flatpak run com.openza.gitdesk
```

<Aside type="tip">
  Prefer AppImage? Both formats are available on the [releases page](https://github.com/openza/gitdesk/releases).
</Aside>
